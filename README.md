hiv0_experiments README!

Folders contain individual .ino files to test individual components.

Hardware Setup:
  1. Chip     = RAK4631
    A. nrf52840
  2. Board    = RAK19001
  3. Screen   = RAK14000 (aka Solomon Systech Ltd. SSD1680: Monochrome E-Paper Screen 2.13" Driver & Controller)
    A. DEPG0213BNS800F42HP
    B. Mono-Color: BLACK WHITE
    C. 2.13 inch
    D. 212*104

Development:
  1. Framework = Arduino
  2. Libraries = Adafruit EPD, Adafruit GFX, Adafruit BusIO.

All original files retain the attribution info from the original creators, if applicable.
